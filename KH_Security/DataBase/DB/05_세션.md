# 세션

- 세션(Session)은 서버가 사용자 상태를 유지하기 위해 사용하는 저장 방식입니다.  
- 사용자마다 고유한 세션 ID(Session ID)를 부여하여 구분합니다.
- 서버 자원을 사용하므로 대규모 데이터를 저장하면 시스템 부담이 커질 수 있습니다.
- 로그인 상태 유지 기능 구현에 자주 사용됩니다.

---

## 동작 원리

- 세션 데이터는 서버에 저장됩니다.
- 클라이언트는 세션 자체가 아닌 세션 ID를 전달합니다.
- 서버는 세션 ID를 이용하여 해당 사용자의 세션 데이터를 조회합니다.
- 세션은 일정 시간 동안 유지되며, 종료되거나 삭제되면 사라집니다.

## 특징
- 서버에서 데이터를 관리하므로 보안성이 높습니다.
- 사용자 수가 많아질수록 서버 자원을 많이 사용합니다.

---

## 세션 관련 함수 정리

- 세션 ID 값을 쿠키로 전달하고 싶지 않는 경우  
php.ini파일에 session.use_cookies 설정 값을 0으로 수정해줍니다.
```php
  <a href="./b.php?<?php echo htmlspecialchars(SID); ?>">b.php</a> // 세션 쿠키를 사용하지 않을 경우 하이퍼링크
```
- 세션 함수는 "서버에 사용자 상태를 저장하고 관리하기 위한 도구"입니다.

### session_start()
- 세션을 시작하는 함수
- 기존 세션이 있으면 불러오고, 없으면 새로 생성
- 반드시 코드의 가장 위(출력 전에) 위치해야 합니다.

---

### $_SESSION
- 세션 데이터를 저장하는 전역 배열
- 사용자 정보 저장 및 조회에 사용
- 예시:
```php
  $_SESSION['id'] = 'user1';   // 저장
  echo $_SESSION['id'];        // 사용
```

---

### session_destroy()
- 현재 세션을 완전히 삭제하는 함수
- 로그아웃 처리 시 사용

---

### session_unset()
- 세션 변수만 초기화 (세션 자체는 유지)

---

### session_id()
- 현재 세션의 ID 값을 확인하거나 설정

---

## 세션을 이용한 인증 처리

- 인증을 위한 테이블 추가를 해주어야합니다.

### md5 함수
- MD5 알고리즘을 이용 해시값을 반환
- 해시값을 생성해주는 PHP 함수입니다.

---

## 세션 예시

- `a.php`
```php
  <?php
     session_start();
     $_SESSION["name"] = '문성기';
     $_SESSION["passwd"] = '암호';
     $name=$_SESSION["name"];
     $passwd= $_SESSION["passwd"];
     echo(" name = $name <br>");
     echo(" passwd = $passwd <br>");
     echo("<a href='./b.php'>b.php</a>");
  ?>
```

- `b.php`
```php
  <?php
     session_start();
     $name=$_SESSION["name"];
     $passwd= $_SESSION["passwd"];
     echo(" name = $name <br>");
     echo(" passwd = $passwd <br>");
     session_destroy();
  ?>
```

- 과정
```
1. a.php에서 session_start()를 실행하여 세션을 시작합니다.
2. $_SESSION["name"]에 '김준혁'을 저장합니다.
3. $_SESSION["passwd"]에 입력한 암호를 저장합니다.
4. 저장된 세션 값을 각각 $name, $passwd 변수에 넣어 출력합니다.
5. b.php 하이퍼링크를 통해 이동합니다.
6. b.php에서도 session_start()를 실행하여 기존 세션에 접근합니다.
7. $_SESSION["name"], $_SESSION["passwd"] 값을 다시 각각 $name, $passwd 변수에 저장 후 해당되는 값을 출력합니다.
8. 마지막에 session_destroy()를 호출하여 세션을 종료합니다.
```
