## 문제 1. 애플리케이션 보안솔루션 설정

### 문제 1-1. 로그인 성공 시 클라이언트의 주소를 저장하는 php코드를 작성하려 한다.
### 클라이언트의 IP주소를 불러오는 슈퍼 글로벌 변수는 무엇인지 기술하세요.

```text
$_SERVER['REMOTE_ADDR']
```

### 문제 1-2. 인증을 위해 사용하는 Session을 저장하기 위한 슈퍼 글로벌 배열은 무엇인지 기술하세요.  

```text
$_SESSION
```

### 문제 1-3. 인증을 위한 세션사용을 위해서 반드시 프로그램 시작에 실행해야 하는 함수는 무엇인지 기술하세요.

```text
session_start();
```

### 문제 1-4. 세션 ID를 쿠키로 전달하지 않도록 php.ini파일에 session.use_cookies 설정 값을 0하면 
### <a herf..></a>에 GET으로 반드시 전달해야하는 무엇인지 기술하세요.

```text
<? echo htmlspecialchars(SID); ?> 또는 htmlspecialchars(SID)
```

## 문제 2. 애플리케이션 보안 신규 위협 대응하기

### 문제 2-1. 인증을 위해 사용가능한 세션과 쿠키중에 쿠키의 보안상의 문제점은 무엇인지 기술하세요.
```text
인증정보가 브라우저에 저장되어 인증정보를 유출할 가능성이 있습니다.
```

### 문제 2-2. 악의적인 스크립트 실행을 방지하기 위해서 스크립트 실행에 사용되는 
### 특수문자나 문자열을 HTML Entities로 바꾸는 함수는 무엇인지 기술하세요.
```text
htmlspecialchars()
```

### 문제 2-3. 악의적인 스크립트 실행을 방지하기 위해서 HTML 및 PHP 태그를 제거하는 함수는 무엇인지 기술하세요.
```text
strip_tags()
```

### 문제 2-4. 다음 인증 데이터 생성용 SQL문의 [?]에 들어갈 내용은 무엇인지 기술하세요.
```text
STANDARD_HASH
```

### 문제 2-5. 위의 SQL문 ('$passwd', 'SHA256') 부분을 보안상 개선하면 어떻게 개선 할 수 있는지 기술하세요.
```text
('$passwd'||'$id', 'SHA256') 또는 ('$id'||'$passwd', 'SHA256')
```
