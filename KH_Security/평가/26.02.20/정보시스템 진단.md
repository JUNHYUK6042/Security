## 문제 1. 단말장치 네트워크 관리

### 문제 1-1. Linux 네트워크 인터페이스(ens160) 설정 파일 내용확인
```text
cat /etc/sysconfig/network-scripts/ifcfg-ens160
```

### 문제 1-2. nmcli 명령으로 ens160의 수정된 설정을 적용하는 명령
```text
nmcli conn up ens160
```

### 문제 1-3. ip 명령으로 현재 활성화된 인터페이스 ens160의 정보만 확인하는 ip 명령

```text
ip a show dev ens160
```

## 문제 5. 백업 및 복구관리

- rsync server의 ip : 192.168.10.11
- rsync server의 서비스명 : backup

### 문제 5-1. Client에서 rsync server의 backup을 /home/backup으로 동기화하기
```text
rsync -avuz 192.168.10.11::backup /home/backup
```

### 문제 5-2. 문제 5-1의 작업을 매일 23시 59분에 자동으로 작업하기위해서 crontab에 추갛라 명령어
```text
59 23 * * * rsync -avuz 192.168.10.11::backup /home/backup
```
