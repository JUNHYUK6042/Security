## 문제 2. 네트워크 스캔 및 서비스 확인

### 문제 2-1. fping을 이용해서 192.168.11.0 네트워크에 활성화된 시스템을 확인
```text
fping -g 192.168.11.0/24
```

### 문제 2-2. 확인된 192.168.11.17을 대상으로 hping3를 이용해서 DNS 서버의 서비스 제공 여부 확인
```text
hping3 -S -p 53 192.168.11.17
```

### 문제 2-3. 서비스가 제공되지 않을 경우 응답
```text
flags=RA 또는 RST + ACK
```

### 문제 2-4. nmap을 이용해서 Stealth 모드로 192.168.11.7의 열려있는 서비스를 탐지하고 확인
```text
nmap -sS 192.168.11.7
```

### 문제 2-5. nmap을 이용해서 192.168.11.7, 192.168.11.17의 서비스를 탐지하고 확인
```text
nmap 192.168.11.7 192.168.11.17
```

### 문제 2-6. nmap을 이용해서 192.168.11.17의 UDP를 이용한 열려있는 서비스를 탐지하고 확인
```
nmap -sU 192.168.11.17
```
