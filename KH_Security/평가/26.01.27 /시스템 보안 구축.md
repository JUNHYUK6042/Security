```text
STP는 looping을 제어하는 프로토콜이다
```

### 문제 3-2. STP의 구현 과정에 대해 기술하세요.

```text
1. 네트워크에서 루트 브리지를 선정합니다.
2. 논루트브리지에서 루트브리지로 가는 하나의 루트 포트를 결정합니다.

4. 나머지 포트를 차단하여 루핑을 방지합니다.
```

## 문제 4. STP에 이용되는 각 용어의 정의를 기술하세요.

### 문제 4-1. Bridge ID

```text
STP에서 스위치를 식별하기위한 정보입니다.
```

### 문제 4-2. Root bridge, Non root bridge

```text
Root bridge : 제일 낮은 BID값을 가진 스위치가 Root bridge가 됩니다.

Non root bridge : root bridge를 제외한 모든 스위치입니다.

```

### 문제 4-3. Root port, Designated port

```text
Root port : non root bridge에서 root bridge까지 가장 빠른 경로에 연결 된 포트입니다.

Designated port : 두 스위치 간의 segment에서 root bridge까지 가는 cost가 가장 낮은 포트입니다.
```
