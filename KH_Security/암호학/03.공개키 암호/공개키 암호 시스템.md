# 공개키 암호 시스템

## 개요

- 공개키 암호 시스템은 **서로 다른 두 개의 키(공개키, 개인키)를 사용하는 암호 방식**입니다.
- 공개키는 누구나 접근 가능하고, 개인키는 소유자만이 보관합니다.
- 기존 관용 암호(대칭키)는 하나의 비밀키를 공유하는 방식입니다.

## 특징

- 암호 알고리즘과 암호키를 알아도 복호키 계산이 불가능합니다.
- 두 개의 키 중 하나는 암호에 다른 하나는 복호에 사용합니다.

---

## 기본 동작 원리

### 암호화

- 송신자가 수신자의 공개키 or 송신자의 개인키(서명문)로 암호화
```
평문 → 수신자 공개키 → 암호문
평문 → 송신자 개인키 → 서명문
```

### 2) 복호화

- 수신자가 자신의 개인키 or 송신자의 공개키로 복호화
```
암호문 → 수신자의 개인키 → 평문
서명문 → 송신자의 공개키 → 평문
```
