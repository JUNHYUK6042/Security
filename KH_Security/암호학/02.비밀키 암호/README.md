# 비밀키 암호

---

## 비밀키 암호의 원리
- 암호화와 복호화에 "같은 키"를 사용하는 방식입니다.

![01]()

### 특징
- 암호 알고리즘의 비도는 키의 비밀성에 의존합니다.
- 관용 암호, 대칭키 암호, 공유키 암호등으로 불립니다.
- 대용량 데이터 암호화에 적합합니다.

---

### 장점
- 속도가 빠릅니다.
  - 실시간 통신, 파일 암호화에 유리합니다.
- 구현이 간단합니다.
  - 시스템 부담이 적습니다.
- 대용량 데이터 처리 가능합니다.

---

### 단점
- 키를 어떻게 안전하게 전달할 것인가에 대한 분배 문제가 있습니다.
- 사용자가 많아지면 키 관리 어려워집니다.
