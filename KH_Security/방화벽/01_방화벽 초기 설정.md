# Astaro 설정

- 방화벽(Firewall)은 내부 네트워크와 외부 네트워크를 분리하고, 그 사이의 트래픽을 통제하는 보안 시스템입니다. 

---

## Astro 설치

### VMware H/W 구성

- Memory : 2G
- NIC : 3개 (INT, EXT, DMZ)
- Hard Disk : 40G 이상

---

- 관리 페이지(WebAdmin)에 접속할 네트워크 인터페이스를 지정하는 설정 단계입니다.

![01](/KH_Security/방화벽/img/01.png)

---

- Network 설정을 다음과 같이 합니다.

![02](/KH_Security/방화벽/img/02.png)

---

- 64bit Kernel 선택은 No로 설정하고, 이후 설치 과정은 기본값으로 모두 Yes를 선택합니다.

![03](/KH_Security/방화벽/img/03.png)

## Astro 초기 설정

### Windows

- Windows 환경에서 브라우저를 열어 `https://192.168.0.1:4444`를 입력합니다.
- 다음과 같이 고급 -> 192.168.0.1(안전하지 않음)(으)로 이동을 클릭합니다.

![07](/KH_Security/방화벽/img/07.png)

---

### 초기 관리자 설정

- 초기 관리자는 다음과 같이 설정합니다.
```
Hostname : fw
Company or Origanization name : itclass
City : sincity
Country : South Korea
admin password : 대문자, 특수문자를 섞어서 입력합니다.
admin account email address : 임의의 이메일을 입력합니다.
```

![08](/KH_Security/방화벽/img/08.png)

- 설정 후 admin과 설정한 Password를 입력합니다.

![09](/KH_Security/방화벽/img/09.png)

## Interface 설정

- 다음 화면은 설정 완료 시 초기 화면입니다.

![19](/KH_Security/방화벽/img/19.png)

---

- 다음 위치로 이동하여 인터페이스를 설정합니다.
- External과 DMZ를 추가합니다.
```
Interfaces & routing -> Interfaces
```

- 다음과 같은 화면이 나옵니다.

![20](/KH_Security/방화벽/img/20.png)


### External 설정
```
name : EXTERNAL
TYPE : Ethernet
Hardware : eth1
IPv4 Address : 192.168.11.254
IPv4 Netmask : /24
IPv4 Default GW : Check
IPv4 Default GW Address : 192.168.11.1
```

---

### DMZ 설정
```
name : EXTERNAL
TYPE : Ethernet
Hardware : eth2
IPv4 Address : 192.168.12.1
IPv4 Netmask : /24
IPv4 Default GW : NO Check
IPv4 Default GW Address : 입력 X
```

- 설정 완료시 다음과 같은 화면이 나오고 down에서 up합니다.
- 방화벽이 어떤 네트워크 구간을 어떻게 나누고 연결하는지 보여주는 핵심 구조입니다.

![21](/KH_Security/방화벽/img/21.png)

- up으로 변경 시 다음과 같은 화면으로 나옵니다.

## Network 구성 추가

- Network 구성은 다음 경로로가서 추가해줍니다.
```
Definition & Users -> Network Definitions -> New Network Definitions
```

- 다음과 같은 화면에서 `in_admin`, `in_oracle`, `ex_main`, `d_web`, `d_dns`를 추가합니다.

![23](/KH_Security/방화벽/img/23.png)

| Name | Type | IPv4 |
| ---- | ---- | ---- |
| in_admin | Host | 192.168.0.3 |
| in_oracle | Host | 192.168.0.11 |
| ex_main | Host | 192.168.11.115 |
| d_web | Host | 192.168.12.11 |
| d_dns | Host | 192.168.12.13 |

![24](/KH_Security/방화벽/img/24.png)

- 네트워크 인터페이스 추가가 되었습니다.
- 방화벽 정책을 쉽게 만들기 위해 IP를 이름으로 추상화한 단계입니다.
