# Packet Tracer
#### Cisco Packer Tracer는 실제 장비 없이도 네트워크를 설계구성 및 실습할 수 있는 시뮬레이션 프로그램입니다
---

### 목차

01) Packet Tracer & CLI 모드 & IP 할당
02) Switch (STP)
03) Switch (VLAN)
04) VLAN Routing & Trunk Routing
05) Router (Static, RIP)
06-1) Router (OSPF)
06-2) Router (OSPF-Multi)
07-1) Router (EIGRP)
07-2) Router (EIGRP - DUAL)
08-1) Redistribute (Static)
08-2) Redistribute (RIP)
08-3) Redistribute (OSPF + EIGRP)
