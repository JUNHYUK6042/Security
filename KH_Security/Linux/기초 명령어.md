# Linux Basic Command

---

## 리눅스 명령어 기본 형식
- 형식 : # command [-단일문자옵션] [--다중문자옵션]
- 예시 : ls -a  -l, ls -al

---

## 디렉터리 경로 표현
| 표기 | 의미 |
|------|------|
| `/`  | 최상위 root 디렉토리 (절대경로 표현의 기준) |
| `.`  | 현재 디렉토리 (상대경로 표현의 기준, 생략 가능) |
| `..` | 현재 디렉토리의 상위 디렉토리 |
| `~`  | 각 사용자의 홈 디렉토리 |

---

## 디렉터리 명령어
| 명령어 | 형식 | 예시 | 설명 |
| ----- | ----- | ----- | ----- |
| `cd` | `cd [디렉터리명]` | `cd /dev` | `해당 디렉터리로 이동` |
| `pwd` | `pwd` | `pwd` | `현재위치 디렉터리경로 출력` |
